1  Installation

This page will guide you in setting up your computer with the R programming environment. This consists of two components: 1) the R programming language and 2) RStudio, an integrated development environment (IDE) for R. Both components are freely available. The R programming language needs to be installed first followed by RStudio. Then, from within RStudio, we’ll install the commonly used tidyverse suite of packages. Finally, we’ll test that everything is running properly.

Note

You will need an active internet connection. We recommend that you close all open programs on your computer before working through the following tasks.

Note

RStudio will change its name to Posit starting October 2022.

1.1 Installing the R programming language

You can install the R programming language on your computer as follows:

Microsoft Windows

Visit https://cran.r-project.org/bin/windows/base/. Skim through the information on that page then, if your system is compatible, click on the link to download R-4.2.1 for Windows (see screenshot below).

Wait until the file fully downloads. Open the downloaded file and follow the instructions to install R on your computer. Use the default options to the extent possible.

MacOS

Visit https://cran.r-project.org/bin/macosx/ and click the relevant .pkg link to download the correct version of R for your computer. If you are on MacOS 10.13 and higher with Intel chip, you should install R-4.2.1.pkg. If you are on Big Sur with an ARM build, you should install R-4.2.1-arm64.pkg. To check the version of your Mac, hit the apple button on the top-left corner of the screen and go to “About This Mac”.

1.2 Installing RStudio

You can install RStudio on your computer by following the steps corresponding to your operating system as follows:

Microsoft Windows

Visit https://www.rstudio.com/products/rstudio/download/. Skim through the information on that page then scroll down (further down!) and click on the link to download RStudio Desktop (see screenshot below):

Run the installer to install RStudio on your computer. Use the default options to the extent possible.

MacOS

Visit https://www.rstudio.com/products/rstudio/download/. Scroll down (further down!) and click on the link corresponding to your macOS operating systems (see screenshot below):

Open the downloaded .dmg file, which should open a folder with icons labeled Applications and RStudio. Drag the RStudio icon into the Applications folder. If you double click Applications, you should now see that RStudio is installed as an application on your computer.

1.3 Installing tidyverse

Open RStudio on your computer (from the start menu in Windows or from the spotlight search utility labeled with the magnifying glass in the top-right corner of the screen in Mac). On a Mac, you may get a warning asking you if you want to open RStudio for the first time, since it was downloaded from the internet; in this case click Open.

Install the tidyverse suite of R packages by running the following command. You run a command by typing it in the console section of RStudio, next to the “>” prompt, and hitting Enter. (Later we’ll see other ways of entering and running R commands). You must be connected to the internet for this to work!

install.packages("tidyverse")
Note

In R, a package is a bundle of R functions (code), documentation on these functions, and (commonly) built-in demo datasets. Packages extend the base R functionality, allowing users to perform certain types of analysis easily and efficiently.

1.4 Testing the installation

To test your installation type in the following command in the Console section of RStudio and hit enter:

data(package = "ggplot2")
